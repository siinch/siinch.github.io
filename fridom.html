ABOUT FRIDOM

Fridom is a physics based game, where the player must avoid obstacles and move the character using gravity. Like Potentiale, Fridom was also made with the Unity game engine.

Fridom started as a winning game-jam game and was finished up and released in 2016.

Originally, there was a mobile version of the game, allowing the player to control the character with the orientation of their phone. This was done using the accellerometer.
